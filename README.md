# Unpaid-Care-Work-Research-Paper
Data analysis research on unpaid care work, focusing on patterns, trends, and socio-economic insights.

## Overview  
This project is based on a research study focused on analyzing unpaid care work and its socio-economic implications. The aim was to understand patterns, trends, and the broader impact of unpaid labour using data-driven insights.

## What I Did  
- Collected and organized relevant data  
- Performed data analysis and interpretation  
- Identified key trends and patterns  
- Presented findings in a structured research format  

## Tools & Skills Used  
- Microsoft Excel  
- Basic statistical analysis  
- Data interpretation  
- Research and analytical thinking  

## Key Takeaway  
This project helped me develop strong attention to detail, analytical thinking, and the ability to work with real-world data to derive meaningful insights.
